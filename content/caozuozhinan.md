# 一、先搞懂：一人公司是什么？

- 法律主体：**一人有限责任公司（自然人独资）**，1 个股东、1 名监事（不可兼任）。
- 核心优势：**有限责任**、可开对公账户、可开票、适合接单 / 创业 / 自媒体 / 技术服务。
- 关键提醒：和个体户、个人独资企业**不同**，一人公司必须**财务独立**，避免无限责任风险。

# 二、7 天落地：注册操作步骤

### 1. 准备材料（1 小时搞定）

- 股东身份证
- 注册地址证明（商用 / 园区挂靠，真实可收件），现在政府鼓励很多园区注册一人公司
- 监事身份证（亲友即可）
- 3–5 个公司名称备选

> 举个例子：
>
> 小李在家创业，没有实际写字楼，本站提供**可用注册地址资源 & 要求**，不用租办公室也能合法注册。

### 2. 线上核名（1–2 天）

- 格式：**地区 + 字号 + 行业 + 有限公司（自然人独资）**
- 渠道：当地政务服务网 → 企业开办一网通办

> 举个例子：
>
> 想叫 “XX 创意设计工作室”，结果核名不通过，本站有**起名 + 核名攻略**，帮你一次通过。

### 3. 设立登记（3–5 天）

- 公司类型：**一人有限责任公司**
- 出资比例：100%
- 组织结构：不设董事会 / 监事会，设**1 名执行董事 + 1 名监事**
- 财务、联络员、代理人均可填本人

> 举个例子：
>

### 4. 刻章 → 银行开户 → 税务报到

1. 刻章：公章、财务章、法人章
2. 开户：必须开**对公账户**，所有业务走公户
3. 税务：30 天内登记，核定税种、申领发票
